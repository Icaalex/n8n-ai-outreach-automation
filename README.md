AI-Powered Customer Outreach Automation

An intelligent, fully automated customer re-engagement 
system built with n8n, Google Gemini AI, and Gmail. 
This workflow runs on a schedule, reads customer data, 
generates hyper-personalized outreach emails using AI, 
and sends them automatically — zero manual effort required.

What It Does:

- Triggers automatically on a set schedule (e.g. every Monday)
- Reads customer data from Google Sheets
- Uses Google Gemini AI to generate a unique, personalized 
  message for each customer based on their name, interests, 
  and last purchase date
- Sends each personalized email via Gmail automatically

Why This Matters:

Manual follow-ups are one of the biggest revenue leaks for 
small businesses. This system eliminates that entirely — 
ensuring every customer gets a timely, personal touchpoint 
without the business owner lifting a finger.

Tech Stack:

- n8n** — workflow automation engine
- Google Gemini AI — LLM for message personalization
- Google Sheets — customer data source
- Gmail — email delivery

Business Impact:

- Saves 3-5 hours/week of manual outreach
- Increases customer retention through consistent follow-up
- Scales to hundreds of customers with no added effort
- Fully customizable for any business niche

How To Use:

1. Clone this repo and import the `.json` file into your n8n instance
2. Connect your Google Sheets, Gmail, and Gemini API credentials
3. Update the Google Sheets URL to your customer list
4. Activate the workflow and let it run
